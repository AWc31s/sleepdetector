# Sleep Detector
A flask web application that checks for closed eyes via the browser camera.

## Server Side Python Dependencies
- numpy
- OpenCV
- Flask
- SocketIO
- Torchvision
- [facenet_models](https://github.com/CogWorksBWSI/facenet_models)
